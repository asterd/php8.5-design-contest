What's New

Built for modern developers

PHP 8.5 brings thoughtful improvements that enhance readability, debugging, and developer experience.

01

Pipe Operator

Chain function calls naturally with the new pipe operator for cleaner, more readable code transformations.

$result = $value |> trim(...) |> strtolower(...);
02

Array First & Last

Simple utility functions to access array boundaries without verbose workarounds or edge case handling.

$first = array_first($array); $last = array_last($array);
03

Fatal Error Backtraces

Stack traces now included for fatal errors, making debugging catastrophic failures significantly easier.

// Full stack trace context // for fatal errors
04

Handler Introspection

Retrieve current error and exception handlers programmatically for better error management tooling.

$handler = get_error_handler(); $exc = get_exception_handler();
05

RTL Locale Detection

Native support for detecting right-to-left writing systems with new internationalization functions.

$isRTL = locale_is_right_to_left('ar');
06

IntlListFormatter

Format lists according to locale conventions with proper grammar and cultural specificity.

// "A, B, and C" or // locale-specific format
07

Memory Limit Ceiling

Set maximum boundaries for memory_limit with the new max_memory_limit INI directive for safer resource management.

max_memory_limit = 512M
08

CLI INI Diff

Quickly identify non-default configuration directives with the new CLI flag for faster debugging.

php --ini=diff
09

Build Date Constant

New PHP_BUILD_DATE constant provides build metadata for version tracking and deployment verification.

echo PHP_BUILD_DATE;
