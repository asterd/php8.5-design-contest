URI Extension

Modern, standards-compliant URL parsing with RFC 3986 and WHATWG URL support

Read RFC →
Before 8.5 
$components = parse_url("https://php.net/releases/8.5/en.php");

var_dump($components['host']);
// string(7) "php.net"
PHP 8.5 
use Uri\Rfc3986\Uri;

$uri = new Uri("https://php.net/releases/8.5/en.php");

var_dump($uri->getHost());
// string(7) "php.net"

Clone with Properties

Modify properties during cloning, enabling elegant immutable patterns with readonly classes

Read RFC →
Before 8.5 
final readonly class PhpVersion {
    public function __construct(
        public string $version = 'PHP 8.4',
    ) {}

    public function withVersion(string $version) {
        return clone($this, ['version' => $version]);
    }
}

$version = new PhpVersion();
var_dump($version->withVersion('PHP 8.5'));
// ✓ Works perfectly!

Pipe Operator

Transform nested function calls into elegant, readable pipelines

Read RFC →
Before 8.5 
$input = ' Some kind of string. ';

$output = strtolower(
    str_replace(['.', '/', '…'], '',
        str_replace(' ', '-',
            trim($input)
        )
    )
);

// Nested, hard to read
PHP 8.5 
$input = ' Some kind of string. ';

$output = $input
    |> trim(...)
    |> (fn (string $s) => str_replace(' ', '-', $s))
    |> (fn (string $s) => str_replace(['.', '/', '…'], '', $s))
    |> strtolower(...);

// Clean, readable flow

#[\NoDiscard] Attribute

Catch bugs at compile-time by marking important return values that shouldn't be ignored

Read RFC →
Before 8.5 
function getPhpVersion(): string {
    return 'PHP 8.4';
}

getPhpVersion(); // No warning

// Return value silently ignored
// Potential bug goes unnoticed
PHP 8.5 
#[\NoDiscard]
function getPhpVersion(): string {
    return 'PHP 8.5';
}

getPhpVersion();
// Warning: The return value should either
// be used or intentionally ignored

Persistent cURL Share Handles

Improved connection reuse and better performance for HTTP requests

Read RFC →
Before 8.5 
$sh = curl_share_init();
curl_share_setopt($sh, CURLSHOPT_SHARE, CURL_LOCK_DATA_DNS);
curl_share_setopt($sh, CURLSHOPT_SHARE, CURL_LOCK_DATA_CONNECT);

$ch1 = curl_init('https://php.net/');
curl_setopt($ch1, CURLOPT_SHARE, $sh);
curl_exec($ch1);

curl_share_close($sh);
curl_close($ch1);
PHP 8.5 
$sh = curl_share_init_persistent([
    CURL_LOCK_DATA_DNS,
    CURL_LOCK_DATA_CONNECT
]);

$ch1 = curl_init('https://php.net/');
curl_setopt($ch1, CURLOPT_SHARE, $sh);
curl_exec($ch1);

// Automatically persisted
curl_close($ch1);

New Array Functions

Simplified array operations with array_first() and array_last()

Read RFC →
Before 8.5 
$php = [
    'php-82' => ['state' => 'security'],
    'php-83' => ['state' => 'active'],
    'php-84' => ['state' => 'active'],
    'php-85' => ['state' => 'upcoming'],
];

$upcomingRelease = null;
foreach ($php as $version) {
    if ($version['state'] === 'upcoming') {
        $upcomingRelease = $version;
        break;
    }
}
PHP 8.5 
$php = [
    'php-82' => ['state' => 'security'],
    'php-83' => ['state' => 'active'],
    'php-84' => ['state' => 'active'],
    'php-85' => ['state' => 'upcoming'],
];

$upcomingRelease = array_first(
    array_filter(
        $php,
        fn($v) => $v['state'] === 'upcoming'
    )
);
