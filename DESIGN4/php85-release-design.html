What's New in PHP 8.5

Major improvements for cleaner code, better performance, and enhanced developer productivity

RFC 3986 and WHATWG URL Compliant API 📄 RFC

New URI extension with standards-compliant URL parsing following RFC 3986 and WHATWG specifications.

PHP < 8.5
$components = parse_url("https://php.net/releases/8.5/en.php");

var_dump($components['host']);
// string(7) "php.net"
PHP 8.5
use Uri\Rfc3986\Uri;

$uri = new Uri("https://php.net/releases/8.5/en.php");

var_dump($uri->getHost());
// string(7) "php.net"
Clone with v2 📄 RFC

Modify properties while cloning objects, including readonly properties, with elegant new syntax.

PHP < 8.5
final readonly class PhpVersion {
    public function __construct(
        public string $version = 'PHP 8.4',
    ) {}

    public function withVersion(string $version) {
        $newObject = clone $this;
        $newObject->version = $version;

        return $newObject;
    }
}

var_dump(new PhpVersion()->withVersion('PHP 8.5'));
// Fatal error: Cannot modify readonly property
PHP 8.5
final readonly class PhpVersion {
    public function __construct(
        public string $version = 'PHP 8.4',
    ) {}

    public function withVersion(string $version) {
        return clone($this, ['version' => $version]);
    }
}

$version = new PhpVersion();

var_dump(
    $version->withVersion('PHP 8.5'),
    $version->version,
);
Pipe Operator 📄 RFC

Chain function calls elegantly with the new pipe operator for more readable code transformations.

PHP < 8.5
$input = ' Some kind of string. ';

$output = strtolower(
    str_replace(['.', '/', '…'], '',
        str_replace(' ', '-',
            trim($input)
        )
    )
);
PHP 8.5
$input = ' Some kind of string. ';

$output = $input
    |> trim(...)
    |> (fn (string $string) => str_replace(' ', '-', $string))
    |> (fn (string $string) => str_replace(['.', '/', '…'], '', $string))
    |> strtolower(...);
New #[\NoDiscard] Attribute 📄 RFC

Mark function return values as important to prevent accidental discarding of critical results.

PHP < 8.5
function getPhpVersion(): string {
    return 'PHP 8.4';
}

getPhpVersion(); // No Errors
PHP 8.5
#[\NoDiscard]
function getPhpVersion(): string {
    return 'PHP 8.4';
}

getPhpVersion();
// Warning: The return value of function getPhpVersion()
// should either be used or intentionally ignored
First Class Callables & Closures in Constant Expressions 📄 RFC 📄 RFC

Use closures and first-class callables in constant expressions, enabling powerful metaprogramming patterns.

PHP < 8.5
final class CalculatorTest extends \PHPUnit\Framework\TestCase
{
    #[DataProvider('subtractionProvider')]
    public function testSubtraction(
        int $minuend,
        int $subtrahend,
        int $result
    ): void
    {
        $this->assertSame(
            $result,
            Calculator::subtract($minuend, $subtrahend)
        );
    }

    public static function subtractionProvider(): iterable
    {
        for ($i = -10; $i <= 10; $i++) {
            yield [$i, $i, 0];
            yield [$i, 0, $i];
            yield [0, $i, -$i];
        }
    }
}
PHP 8.5
final class CalculatorTest
{
    #[Test\CaseGenerator(static function(): iterable {
        for ($i = -10; $i <= 10; $i++) {
            yield [$i, $i, 0];
            yield [$i, 0, $i];
            yield [0, $i, ($i * -1)];
        }
    })]
    public function testSubtraction(
        int $minuend,
        int $subtrahend,
        int $result
    )
    {
        \assert(
            Calculator::subtract($minuend, $subtrahend) === $result
        );
    }
}
Persistent cURL Share Handle Improvement 📄 RFC 📄 RFC

Simplified persistent cURL share handles with new initialization function and automatic resource management.

PHP < 8.5
$sh = curl_share_init();
curl_share_setopt($sh, CURLSHOPT_SHARE, CURL_LOCK_DATA_DNS);
curl_share_setopt($sh, CURLSHOPT_SHARE, CURL_LOCK_DATA_CONNECT);

curl_share_close($sh);

curl_close($ch2);

New array_first() and array_last() Functions 📄 RFC

Convenient new array utility functions to get the first or last element with cleaner, more expressive code.

PHP < 8.5
$php = [
    'php-82' => ['state' => 'security', 'branch' => 'PHP-8.2'],
    'php-83' => ['state' => 'active', 'branch' => 'PHP-8.3'],
    'php-84' => ['state' => 'active', 'branch' => 'PHP-8.4'],
    'php-85' => ['state' => 'upcoming', 'branch' => 'PHP-8.5'],
];

$upcomingRelease = null;
foreach ($php as $key => $version) {
    if ($version['state'] === 'upcoming') {
        $upcomingRelease = $version;
        break;
    }
}

var_dump($upcomingRelease);
PHP 8.5
$php = [
    'php-82' => ['state' => 'security', 'branch' => 'PHP-8.2'],
    'php-83' => ['state' => 'active', 'branch' => 'PHP-8.3'],
    'php-84' => ['state' => 'active', 'branch' => 'PHP-8.4'],
    'php-85' => ['state' => 'upcoming', 'branch' => 'PHP-8.5'],
];

$upcomingRelease = array_first(
    array_filter(
        $php,
        static fn($version) => $version['state'] === 'upcoming'
    )
);

var_dump($upcomingRelease);

Additional Features & Improvements

More enhancements to make PHP development smoother and more powerful

Property Promotion for Final Classes

Property promotion is now available for final classes, reducing boilerplate code even further.

Attributes for Constants

Attributes are now available for constants, enabling better metadata and documentation capabilities.

#[\Override] for Properties

The #[\Override] attribute now works on properties, improving type safety in inheritance hierarchies.

#[\Deprecated] for Traits

Mark traits as deprecated with the #[\Deprecated] attribute to guide developers away from outdated code patterns.

Asymmetric Visibility for Static Properties

Control read and write access separately for static properties, enhancing encapsulation in class design.

#[\DelayedTargetValidation]

New #[\DelayedTargetValidation] attribute for more flexible attribute validation timing.

Error Handler Introspection

New get_error_handler(), get_exception_handler() functions, and Closure::getCurrent method for better debugging.

Enhanced DOM Methods

New Dom\Element::getElementsByClassName() and Dom\Element::insertAdjacentHTML() methods for easier DOM manipulation.

Enchant Dictionary Functions

New enchant_dict_remove_from_session() and enchant_dict_remove() functions for spell-checking control.

Grapheme Levenshtein

New grapheme_levenshtein() function for Unicode-aware string distance calculations.

OPcache File Cache Check

New opcache_is_script_cached_in_file_cache() function to verify file cache status.

Reflection Enhancements

New methods: ReflectionConstant::getFileName(), getExtension(), getExtensionName(), getAttributes(), and ReflectionProperty::getMangledName().

cURL Multi Handle Info

New curl_multi_get_handles() function to retrieve all handles from a multi handle.

CurlSharePersistentHandle Class

New CurlSharePersistentHandle class for better resource management in persistent connections.

URI Extension Classes

Full suite of URI classes: Uri\UriException, Uri\InvalidUriException, Uri\UriComparisonMode, Uri\Rfc3986\Uri, Uri\WhatWg\Url and related classes.

Deprecations and Backward Compatibility

PHP 8.5 maintains excellent backward compatibility with minimal breaking changes. This release focuses on adding new features while preserving existing functionality. Detailed migration information is available in the official PHP 8.5 migration guide.
