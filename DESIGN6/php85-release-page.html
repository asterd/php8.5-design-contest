API URI conforme a RFC 3986 & WHATWG

RFC
PHP < 8.5
$components = parse_url(
    "https://php.net/releases/8.5/en.php"
);

var_dump($components['host']);
// string(7) "php.net"
→
PHP 8.5
use Uri\Rfc3986\Uri;

$uri = new Uri(
    "https://php.net/releases/8.5/en.php"
);

var_dump($uri->getHost());
// string(7) "php.net"

Clonazione con Modifica di Proprietà

RFC
PHP < 8.5
final readonly class PhpVersion {
    public function __construct(
        public string $version = 'PHP 8.4',
    ) {}

    public function withVersion(string $version) {
        $newObject = clone $this;
        $newObject->version = $version;
        return $newObject;
    }
}
// Error: Cannot modify readonly property
→
PHP 8.5
final readonly class PhpVersion {
    public function __construct(
        public string $version = 'PHP 8.4',
    ) {}

    public function withVersion(string $version) {
        return clone($this, [
            'version' => $version
        ]);
    }
}
// Funziona con proprietà readonly!

Operatore Pipe per Trasformazioni Leggibili

RFC
PHP < 8.5
$input = ' Some kind of string. ';

$output = strtolower(
    str_replace(['.', '/'], '',
        str_replace(' ', '-',
            trim($input)
        )
    )
);
// Annidato, difficile da leggere
→
PHP 8.5
$input = ' Some kind of string. ';

$output = $input
    |> trim(...)
    |> (fn ($s) => str_replace(' ', '-', $s))
    |> (fn ($s) => str_replace(['.', '/'], '', $s))
    |> strtolower(...);
// Pulito, lineare, leggibile!

Attributo #[\NoDiscard]

RFC
PHP < 8.5
function getPhpVersion(): string {
    return 'PHP 8.4';
}

getPhpVersion();
// Nessun warning
→
PHP 8.5
#[\NoDiscard]
function getPhpVersion(): string {
    return 'PHP 8.5';
}

getPhpVersion();
// Warning: Return value should be used

Nuove Funzioni array_first() e array_last()

RFC
PHP < 8.5
$php = [
    'php-82' => ['state' => 'security'],
    'php-85' => ['state' => 'upcoming'],
];

$result = null;
foreach ($php as $v) {
    if ($v['state'] === 'upcoming') {
        $result = $v;
        break;
    }
}
→
PHP 8.5
$php = [
    'php-82' => ['state' => 'security'],
    'php-85' => ['state' => 'upcoming'],
];

$result = array_first(
    array_filter(
        $php,
        fn($v) => $v['state'] === 'upcoming'
    )
);
